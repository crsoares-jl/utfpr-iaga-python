\# Fundamentos de Python



Repositório desenvolvido durante a disciplina \*\*Fundamentos de Python\*\* da Especialização em Inteligência Artificial Generativa da UTFPR.



\## Objetivo



Este repositório reúne os exercícios, exemplos e projetos desenvolvidos ao longo da disciplina, servindo como material de estudo e referência para futuras aplicações em Inteligência Artificial.



\## Conteúdo



Atualmente o projeto contém:



\- Exercícios de lógica em Python;

\- Manipulação de listas, tuplas e dicionários;

\- Funções e módulos;

\- Leitura e gravação de arquivos;

\- Consumo de APIs;

\- Organização de projetos Python;

\- Outros exemplos desenvolvidos durante o curso.



\## Tecnologias utilizadas



\- Python 3.14

\- Git

\- GitHub

\- Visual Studio Code / PyCharm

\## Como clonar o projeto



```bash

git clone https://github.com/crsoares-jl/utfpr-iaga-python.git

cd utfpr-iaga-python

```



\## Criando um ambiente virtual



\### Windows



```bash

python -m venv .venv

.venv\\Scripts\\activate

```



\### Linux



```bash

python3 -m venv .venv

source .venv/bin/activate

```



\## Instalação das dependências



```bash

pip install -r requirements.txt

```



\## Executando um exemplo



```bash

python nome\_do\_arquivo.py

```



\## Organização



O projeto segue uma estrutura modular, permitindo que novas aulas e exercícios sejam adicionados facilmente.
